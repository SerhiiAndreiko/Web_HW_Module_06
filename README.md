# Університетська База Даних

Цей скрипт створює базу даних університету та заповнює її випадковими даними за допомогою бібліотеки `faker`.

## Використання

1. Встановіть необхідні бібліотеки за допомогою команд:

```bash
pip install faker sqlite3


Запустіть скрипт generate_data.py:
python generate_data.py

Цей скрипт виконує наступні дії:

Створює та встановлює з'єднання з базою даних у файлі university.db.

Додає загальні назви груп до таблиці groups.

Додає імена та прив'язує їх до групи до таблиці students.

Додає загальні імена викладачів до таблиці teachers.

Додає загальні предмети та прив'язує їх до викладачів до таблиці subjects.

Генерує випадкові оцінки для кожного студента з кожного предмету та додає їх до таблиці grades.

Результати будуть збережені у файлі university.db та можуть бути використані для подальших запитів.
Опис таблиць
groups: назви груп.

students: інформація про студентів, включаючи ім'я та групу.

teachers: інформація про викладачів.

subjects: інформація про предмети та викладачів, які їх викладають.

grades: оцінки студентів з предметів.

Приклад використання запиту
Для знаходження 5 студентів із найвищим середнім балом з усіх предметів використовуйте наступний SQL-запит:

query_01.sql
select s.student_id, s.student_name, avg(g.grade) as avg_grade
from students s
join grades g on s.student_id = g.student_id
group by s.student_id, s.student_name
order by avg_grade desc
limit 5;

Цей запит виведе імена та середні бали п'яти студентів з найвищим середнім балом з усіх предметів.
